## Platform Identity

**BPOC** = Careers Platform (Core Infrastructure)

BPOC is the underlying careers platform that powers recruitment for agencies and their clients. It provides:

- Candidate talent pool
- Job posting infrastructure
- Video interview system (Daily.co)
- Application tracking
- Offer management
- API for whitelabeled portals

---

## User Roles

| Role | Description | Access Level |
|------|-------------|--------------|
| **Candidate** | Job seekers who sign up looking for work | Apply to jobs, manage profile |
| **Recruiter** | Attached to Agency, manages recruitment flow | Full recruitment management |
| **Client** | Companies with dashboard - post jobs, browse talent, hire | Full hiring capabilities |
| **Admin** | BPOC internal team | Platform oversight & management |

## Client Capabilities

Clients have **FULL ACCESS** to talent and hiring within the Agency Portal.

### Client Dashboard Features

| Feature | Description |
|---------|-------------|
| **Post Jobs** | Create job listings attached to Agency |
| **View Talent Pool** | Browse ALL candidates in the system |
| **Search & Filter** | Find candidates by skills, experience, location |
| **View Profiles** | See full candidate profiles, resumes, assessments |
| **Request Interview** | Directly request interview with ANY candidate |
| **Skip Process** | Bypass normal application/screening flow if desired |
| **Schedule Interviews** | Book interview times with candidates |
| **Send Offers** | Create and send job offers |
| **Hire Candidates** | Complete the hiring process |

### Gate Database Fields

| Field | Type | Description |
|-------|------|-------------|
| `released_to_client` | BOOLEAN | When TRUE, client can see application |
| `released_at` | TIMESTAMP | When application was released |
| `released_by` | UUID | Recruiter who released |
| `video_call_rooms.share_with_client` | BOOLEAN | Per-call toggle: when TRUE, client can see ALL artifacts for that call (recording/transcript/notes) |
| `video_call_rooms.share_with_candidate` | BOOLEAN | Per-call toggle: when TRUE, candidate can see ALL artifacts for that call (recording/transcript/notes) |

### Important Note

> **The Recruiter Gate ONLY applies to the Normal Application Flow.**
> 
> When clients use the **Direct Talent Pool** path, they bypass the gate entirely and can request interviews with any candidate.
